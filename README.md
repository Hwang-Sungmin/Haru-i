# Haru-i (하루-아이) 👶📸

**Haru-i**는 AI 기반의 안드로이드 네이티브 아기 사진 관리 앱입니다. 기기 내 수많은 사진 중 우리 아기 사진만 정교하게 찾아내고, 소중한 순간들을 기록하며 예쁘게 꾸밀 수 있는 기능을 제공합니다.

## 🚀 주요 기능

### 1. 스마트 갤러리 & 타임라인 (Custom Selection)
*   **연도-월별 필터링**: 기존의 긴 나열 방식에서 탈피하여, 헤더의 연도-월을 클릭해 원하는 시점의 사진만 쏙쏙 골라볼 수 있는 인터랙티브 타임라인을 제공합니다.
*   **최신순 자동 정렬**: 앱 실행 시 가장 최신의 추억을 먼저 보여주어 탐색의 번거로움을 줄였습니다.

### 2. 고도화된 아기 사진 판별 (AI Smart Filter)
*   **온디바이스 분석 (ML Kit)**: 기기 내에서 1차적으로 인물 얼굴과 아기 관련 라벨(`Baby`, `Infant` 등)을 분석합니다.
*   **서버 기반 정밀 인식 (DeepFace)**: 특정 인물(우리 아기)만 정확히 인식하기 위해 Python AI 서버와 연동하여 얼굴 지문(Embedding) 비교 분석을 수행합니다.

### 3. 하이라이트 & 앨범 그룹화 (Room DB)
*   **앨범 폴더링**: 홈 화면의 앱 아이콘을 묶는 것처럼, 하이라이트 사진들을 주제별 폴더(앨범)로 묶어 관리할 수 있습니다.
*   **2x2 미리보기**: 폴더 아이콘 내부에 포함된 사진들을 미리 볼 수 있어 직관적인 탐색이 가능합니다.

### 4. 감성 사진 꾸미기 (Photo Edit)
*   **펜 드로잉**: 다양한 색상의 펜으로 사진 위에 직접 낙서하거나 메모를 그릴 수 있습니다.
*   **스티커 & 텍스트**: 귀여운 이모지 스티커와 텍스트 문구를 사진 위에 배치하여 성장 카드를 만듭니다.
*   **편집 저장**: 꾸민 결과물을 원본 해상도에 맞춰 기기 갤러리(`Pictures/Harui`)에 새 파일로 저장합니다.

### 5. 따뜻한 감성 테마 (Material 3)
*   **웜톤 팔레트**: 베이지, 소프트 코랄, 크림색 중심의 따뜻한 컬러 테마를 적용하여 아기 사진첩에 어울리는 포근한 분위기를 연출했습니다.

## 🛠 기술 스택

### Android App
*   **Language**: Kotlin (2.0.21)
*   **UI Framework**: Jetpack Compose (Material 3)
*   **Database**: Room DB (KSP 기반, v2)
*   **AI/ML**: Google ML Kit (On-device)
*   **Image Loading**: Coil

### AI Server
*   **Language**: Python 3.10+
*   **Framework**: FastAPI
*   **AI Library**: DeepFace, TensorFlow, OpenCV
*   **Model**: VGG-Face (Face Verification)

## 📱 실행 및 설정
*   **서버 실행**: `server/app/main.py` 실행 (포트 8000)
*   **권한**: `READ_MEDIA_IMAGES` (Android 13+), `READ_EXTERNAL_STORAGE` (Android 12-)
*   **저장 경로**: 편집된 사진은 기기 내 `Pictures/Harui` 폴더에 저장됩니다.
