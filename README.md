# Haru-i (하루-아이) 👶📸

**Haru-i**는 AI 기반의 안드로이드 네이티브 아기 사진 관리 앱입니다. 기기 내 수많은 사진 중 우리 아기 사진만 정교하게 찾아내고, 소중한 순간들을 기록하며 예쁘게 꾸밀 수 있는 기능을 제공합니다.

---

## 🚀 주요 기능

### 1. 스마트 갤러리 & 타임라인
*   **연도-월별 필터링**: 헤더의 연도-월을 클릭해 원하는 시점의 사진만 쏙쏙 골라볼 수 있는 인터랙티브 타임라인.
*   **최신순 자동 정렬**: 앱 실행 시 가장 최신의 추억을 먼저 노출.

### 2. 고도화된 아기 사진 판별 (AI Hybrid Filter)
*   **1단계 (온디바이스)**: Google ML Kit을 통해 얼굴 유무 및 아기 관련 라벨(`Baby`, `Infant` 등) 1차 선별.
*   **2단계 (서버 정밀 인식)**: Python AI 서버(DeepFace + MediaPipe)와 연동하여 등록된 '우리 아기'와 얼굴 지문(Embedding) 비교 분석.

### 3. 하이라이트 & 앨범 그룹화
*   **앨범 폴더링**: 사진들을 주제별 폴더로 묶어 관리 (앱 아이콘 폴더링 방식).
*   **2x2 미리보기**: 폴더 아이콘에서 내부 사진들을 미리 볼 수 있는 직관적인 뷰.

### 4. 감성 사진 꾸미기 & 저장
*   **편집 도구**: 펜 드로잉, 스티커(이모지), 텍스트 추가 기능 제공.
*   **고해상도 저장**: 꾸민 내용을 원본 비트맵 좌표에 정확히 매핑하여 `Pictures/Harui` 폴더에 저장.

### 5. AI 분석 모니터링 대시보드
*   **웹 기반 모니터링**: 서버 구동 시 브라우저를 통해 현재 분석 중인 사진과 과거 이력을 한눈에 확인 가능.
*   **시스템 제어**: 대시보드에서 분석 중단/재개 및 시스템 상태를 실시간으로 제어.

## 🖥 AI 분석 서버 사용 가이드 (Python)

정밀 인물 인식을 위해 별도의 서버 실행이 필요합니다.

### 1. 서버 환경 구축
```bash
# 1. 서버 폴더 이동
cd server

# 2. 필수 라이브러리 설치
pip3 install -r requirements.txt

# 3. 서버 실행
python3 app/main.py
```
*   **서버 주소**: 기본적으로 `http://localhost:8000`에서 대기합니다. 브라우저로 접속하면 **AI 분석 대시보드**를 볼 수 있습니다.

### 2. 서버 파일 구조 (Refactored)
코드가 기능별로 분리되어 관리가 용이합니다.
*   `app/main.py`: 서버 메인 진입점 및 핵심 API 엔드포인트 관리.
*   `app/dashboard.py`: 웹 대시보드 UI 및 루트 경로 관련 로직.
*   `app/state.py`: 분석 히스토리 및 시스템 상태 데이터 중앙 관리.

## 🧪 우리 아기 정밀 분석 테스트 방법

### Step 1: 우리 아기 기준 사진 등록
1.  앱 우측 상단 **설정(톱니바퀴)** 클릭.
2.  상단 **프로필 영역**을 터치하여 갤러리에서 우리 아기의 **선명한 정면 사진** 선택.
3.  이름 입력 후 **저장** 버튼 클릭.

### Step 2: 정밀 분석 실행 및 모니터링
1.  **타임라인** 탭에서 분석하고 싶은 **월**을 선택.
2.  **'아기 사진 찾기'** 버튼 클릭.
3.  **대시보드 확인**: 브라우저에서 `http://localhost:8000`에 접속하여 서버가 사진을 어떻게 처리하고 있는지 실시간으로 확인하세요.
    *   **현재 분석 사진**: 실시간으로 처리 중인 이미지가 표시됩니다.
    *   **분석 히스토리**: 결과(MATCH/NO MATCH)와 유사도 점수를 테이블로 보여줍니다.

## 🛠 기술 스택

### Android App
*   **Kotlin (2.0.21)**, **Jetpack Compose (Material 3)**
*   **Room DB** (앨범 및 메타데이터 저장)
*   **Retrofit2 & OkHttp** (AI 서버 통신)
*   **Google ML Kit** (On-device Face/Label Detection)

### AI Server
*   **Python 3.10+**, **FastAPI**
*   **DeepFace** (Facenet512 모델)
*   **MediaPipe / SSD / OpenCV** (Face Detection Backends)
